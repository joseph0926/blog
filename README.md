## 실행

```sh
# 빌드
pnpm build
# 실행
pnpm dev
# blog(next.js): localhost:3000
# planner(react-router): localhost:5173

# 체크
pnpm lint:fix
pnpm format
pnpm type-check

# DB
pnpm db:gen
```
